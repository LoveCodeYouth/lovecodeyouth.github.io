Mysql之DDL

发表于2023-09-15|更新于2023-09-17
|阅读量:

DDL–数据库操作

　　查询：
　　1.查询所有库：show databases;
　　2.查询当前数据库：select database();
　　3.创建：create database[if not exists]数据库名[default charset字符集（例：UTF-8MD4）][collate排序规则]；
　　4.删除数据库：drop database[if exists]数据库名；
　　5.使用数据库：use 数据库名;

DDL–表操作-查询

　　1.查询当前数据库中的所有表：show tables;
　　2.查询表的结构：desc 表名；
　　3.查询指定表的建表语句： show create table 表名；

DDL–表操作-创建表

　　creat table 表名（
　　字段1 字段1类型[comment 字段1注释],
　　字段2 字段2类型[comment 字段2注释],
　　字段3 字段3类型[comment 字段3注释],
　　…… 　　字段n 字段n类型[comment 字段n注释] 　　）[comment 表注释]; 　　 　　例：create table t_user(`
　　id int comment '编号'，
　　name varchar(50) comment '姓名'，
　　age int comment '年龄',
　　gender varchar(1) comment '性别'
　　)comment '用户信息';

DDL–表操作-数据类型

　　主要分为三大类：数值类型，字符串类型，日期时间类型。

DDL–表操作-修改

　　1.添加字段：alter table 表名 add 字段名 类型（长度） [comment 注释] [约束];
　　例如：为emp表添加一个新字段“昵称”，为nickname,类型为varchar（20）
　　 　　alter table emp add nickname varchar(20) comment '昵称';
　　 alter table usinfo add nickname varchar(20) comment'测试';
　　2.修改字段名和字段类型： alter table 表名 change 旧字段名 新字段名 类型(长度) [comment 注释] [约束];
　　 例： 将emp表的nickname 字段 修改为usename ,类型为varchar(30)
　　
　　3.删除字段:alter table 表名 drop 字段名;
　　 例：alter table usinfo drop nickname;
　　
　　4.修改表的名字：alter table 表名 rename to 新的表名;

DDL-表操作-删除

　　1.删除表:drop table [if exists] 表名;
　　2.删除指定表，并重新创建该表： truncate table 表名；
